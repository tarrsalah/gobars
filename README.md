gobars
======

https://github.com/jamesward/jaxrsbars demo in go

### Build
  * Be sure that `gobars` is on your `$GOPATH/src`
  * `cd $GOPATH/src/gobars`
  * `./go build`
  * `./gobars`
  * Open your brower on `localhost:8081`
